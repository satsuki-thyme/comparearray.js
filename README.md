# comparearray.js
配列を比較する JavaScript のライブラリです。

## 目次
- comparearray.js  
  ライブラリ本体。
- comparearray.min.js  
  ライブラリ本体。圧縮版。
- sample.html  
  使い方のサンプル。
- comparearray.html  
  応用編。

## 説明
- 第 1 引数と第 2 引数に比較する配列または連想配列を与えます。
- 戻り値は数値です。
  - 0: 構造も値も等しい
  - 1: 構造は等しいが値が異なる
  - 2: 構造が異なる
